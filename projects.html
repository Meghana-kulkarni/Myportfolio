🟢 Task Management System

Problem: Managing daily tasks manually becomes difficult due to lack of organization, task tracking, and secure user-based access.

Solution: Developed a full-stack MERN Task Management application that enables users to securely manage, track, and organize tasks through an interactive dashboard.

Key Features:

  • User Registration & Login Authentication
  • JWT-based Protected Routes
  • Create, Edit, Delete Tasks
  • Task Status Toggle (Todo / Done)
  • Task Filtering (All / Pending / Completed)
  • Responsive and Modern UI Design

Tech Stack: React.js, Node.js, Express.js, MongoDB Atlas, JWT, Tailwind CSS, Axios, Render, Vercel

My Contributions:

  • Designed and implemented REST APIs for task management
  • Built secure authentication and authorization using JWT
  • Integrated MongoDB database with backend services
  • Developed responsive frontend UI using React and Tailwind CSS
  • Implemented task filtering, editing, and status management features
  • Deployed frontend and backend on cloud platforms

Impact:

  • Simplified task organization and tracking process
  • Improved productivity through centralized task management
  • Enhanced understanding of full-stack development and deployment workflows
